Minimal API

Drop‑in gate with smart energy control.

from sundew import SundewAlgorithm from sundew.config import SundewConfig cfg = SundewConfig(activation_threshold=0.78, target_activation_rate=0.15, gate_temperature=0.08, max_threshold=0.92, energy_pressure=0.04) algo = SundewAlgorithm(cfg) x = {"magnitude": 63, "anomaly_score": 0.52, "context_relevance": 0.31, "urgency": 0.18} res = algo.process(x) print(res or "Skipped") print(algo.report())

API cheatsheet

  • SundewConfig: all knobs; call validate() for sanity checks.
  • SundewAlgorithm(cfg): gate/controller instance.
  • algo.process(x) → ProcessingResult | None.
  • algo.report() → metrics dict.
  • Input keys: magnitude (0–100), anomaly_score, context_relevance, urgency ∈ [0,1].
  • Useful knobs: target_activation_rate, gate_temperature, max_threshold, energy_pressure, probe_every, refractory.
