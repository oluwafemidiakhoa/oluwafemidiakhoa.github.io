What is Sundew?

Sundew is an adaptive gating algorithm inspired by the sundew plant: it decides when to fully process an input and when to skip, trading a tiny accuracy drop for large energy savings. It tunes its threshold online using a lightweight PI controller, respects an energy budget, and supports “soft” logistic gating via temperature.

Ideal for edge devices, streaming analytics, and any scenario where every millijoule or CPU cycle matters.
